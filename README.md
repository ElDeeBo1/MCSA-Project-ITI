# Windows Server System Administration Project

A comprehensive Windows Server project covering Active Directory, DHCP, Group Policy, Storage, and Network Services implementation.

---

## Project Overview

Designed and implemented a full enterprise IT infrastructure using Windows Server, including domain services, user management, network configuration, and security policies.

---

## Key Implementations

- Configured Primary Domain Controller (PDC) for `ITISYSADMIN.com` and added Child Domain Controller
- Created Organizational Units (OUs), users, and groups for IT, HR, and Sales departments with delegated administrative control
- Implemented DHCP with IP range, reservations, MAC filtering, and backup configuration
- Applied security policies using Group Policy (GPO): drive restrictions, wallpaper customization, startup scripts
- Configured shared storage with role-based permissions and disk quotas
- Set logon hours and restricted users to specific machines
- Implemented VPN and RRAS for remote access
- Created different storage volumes (Spanned, Striped, Fault-tolerant)

---

## Technologies Used

- Windows Server
- Active Directory (AD DS)
- DHCP
- Group Policy (GPO)
- RRAS & VPN
- NTFS Permissions & Disk Management

---

## Project Goal

To simulate a real-world enterprise environment and practice system administration, network configuration, and security management.
